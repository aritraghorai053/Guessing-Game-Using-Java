# 🎯 Number Guessing Game (Java)

A simple console-based Number Guessing Game built using Java. The program randomly generates a number, and the player has to guess it within multiple attempts. After each guess, the game provides hints whether the guessed number is too high or too low until the correct number is found.

## 🚀 Features

- 🎲 Random number generation
- 🎮 Interactive console gameplay
- 📈 Hint system (Too High / Too Low)
- 🔢 Tracks number of attempts
- ✅ Input validation (if implemented)
- 💻 Beginner-friendly Java project

## 🛠️ Technologies Used

- Java
- OOP Concepts
- Scanner Class
- Random Class

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/GuessingGame.git
```

2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).

3. Compile the program

```bash
javac GuessingGame.java
```

4. Run the program

```bash
java GuessingGame
```
